# 🎯 可见构件导出工具 V2

## 📖 项目简介

Revit 可见构件导出工具，支持一键匹配计量参数、智能过滤和导出可见构件到 Excel。

**当前版本**: V2.2.1  
**状态**: ✅ 已修复，等待测试

---

## 🚀 快速开始

### 安装
插件已编译并安装到 Revit 2024

### 使用
1. 打开 Revit 项目
2. 切换到 3D 视图
3. 点击"一键匹配计量参数"
4. 点击"导出可见构件"

## 技术路线
- **开发语言**: C# (Revit API)
- **数据处理**: 提取 `ActiveView` 下的 `FilteredElementCollector`。
- **导出方式**: 采用 `ClosedXML` 库实现无后台 Excel 生成，确保运行效率。
- **输出位置**: 默认导出至用户桌面，文件命名规则：`BIM构件导出_时间戳.xlsx`。
